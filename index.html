🚀 วิธีสร้าง Firebase

1
ไปที่ console.firebase.google.com → สร้าง Project ใหม่ (ฟรี)
2
Build → Realtime Database → Create database
เลือก Start in test mode
3
Project Settings ⚙️ → General → Your apps
คลิก </> Web → Register app → เอา config มาวางด้านล่าง
4
วาง firebaseConfig ทั้งก้อน (JSON) ในช่องด้านล่าง
ทุกคนในบ้านต้องใช้ Room Key เดียวกัน
